Install binaries from Bioconductor

Binary installation is available through the Bioconductor package repository. Binary installation of the current release version (1.8) requires R 3.5.0 and Bioconductor version 3.7:

## try http:// if https:// URLs are not supported
source("https://bioconductor.org/biocLite.R")
biocLite("dada2")

Allow installation of any additional packages requested. You are ready to start using dada2!

If the above installed an early version of dada2 (e.g. 1.4 or 1.6) you can get the current dada2 release by updating to Bioconductor 3.7: biocLite("BiocUpgrade") then biocLite(). This may require updating R as well.
